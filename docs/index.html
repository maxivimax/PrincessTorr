PrincessTorr

Kodi addon: a TMDb catalog of movies and TV shows, on-demand torrent search and playback via Jacktorr/TorrServer. Native UI - your skin drives the look.

Add to Kodi

Install the repository once - Kodi keeps the addon updated automatically.

  1. Kodi → Settings → System → Add-ons → enable Unknown sources.
  2. Download the repository zip (button below).
  3. Add-ons → Install from zip file → pick the downloaded zip.
  4. Add-ons → Install from repository → PrincessTorr Repository → Video add-ons → PrincessTorr.
Download repository zip

Repository index: https://maxivimax.github.io/PrincessTorr/addons.xml

Features

  • TMDb catalog with localized metadata (Russian by default): movies, TV shows, genres, search.
  • On-demand torrent search - JacRed (public RU tracker aggregator, like in Lampa) or Prowlarr.
  • Voice/dub, quality, size and seeders right in the release line.
  • List sorting and filters: seeders/size/quality, Russian only, search by word.
  • Pick the list view (posters, wall, …) and make it the default per skin.
  • Episode air status, continue watching and local history.
  • Playback via Jacktorr. Search runs only when you open a title, so the catalog opens fast.

Configuration

  • TMDb: API key (v3 or v4 Bearer) and metadata language.
  • Search source: JacRed (default) or Prowlarr with address and key.
  • Network: HTTP proxy if needed.

Needs plugin.video.jacktorr pointed at your TorrServer. If TMDb is DNS-blocked, a clean DNS on the device helps.
